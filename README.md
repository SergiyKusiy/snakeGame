# Snake Game

A Snake game built with Pixi.js and vanilla JavaScript.

## Game Modes

- **Classic** — the game ends if the snake hits the wall or its own body
- **No Die** — the snake passes through walls and its own body
- **Walls** — each eaten food spawns a new wall on the field
- **Portal** — two food items on the field; eating one teleports the snake to the other
- **Speed** — the snake speeds up by 10% with each eaten food

## Controls

Arrow keys to move the snake.

## Getting Started

```bash
npm install
npm run dev
```
